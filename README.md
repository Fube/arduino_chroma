# Chroma

## It kinda works

### Parts Needed

-   Arduino (I used an UNO)
-   300Ω resistor x 3
-   1kΩ resistor x 2
-   Male-to-Female jumper wires x 4
-   Male-to-Male jumper wires x 10
-   RGB LED
-   Joystick
-   Breadboard

### Usage Notes

When uploading, do not touch the Joystick, it'll mess up the color calculations.
